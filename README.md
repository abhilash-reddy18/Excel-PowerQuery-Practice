# Excel Power Query Practice

This repository contains my hands-on practice work using Microsoft Excel Power Query.

I worked on multiple real-world scenarios including importing data from different sources, cleaning and transforming data, and combining datasets for analysis.

## Topics Covered
- Importing data from Web, PDF, CSV, and Excel files  
- Appending data from multiple sheets and workbooks  
- Handling large datasets efficiently  
- Cleaning junk rows and formatting issues  
- Using delimiters for structured data  
- Advanced grouping and transformations  
- Real-world job-style Power Query use cases  

## Tools Used
- Microsoft Excel  
- Power Query
